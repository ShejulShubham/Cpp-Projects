# To Run the codes
 ### - Open terminal and run following command
 1. To compile the code
    ```BASH
    g++ fileName.cpp
    ```

2. If you want custom filename
     ```BASH
     g++ filename.cpp -o output_filename
     ```

3. To execute the code
    ```BASH
    # For Linux
        ./a.out
    ```

    ```CommandPrompt
    # For Windows
        ./a.exe
    ```
4. For custom filename
    ```BASH
    # For Linux
        ./output_filename.out
    ```

    ```CommandPrompt
    # For Windows
        ./output_filename.exe
    ```
