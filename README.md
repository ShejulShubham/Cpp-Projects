# To Run the codes
 ### - Open terminal and run following command
 1. To compile the code
    ```BASH
    g++ fileName.cpp
    ```
1.1 If you want custom filename
    ```BASH
        # For both OS
        g++ filename.cpp -o output_filename
    ```

 2. To execute the code
    ```BASH
    # For Linux
        ./a.out
    ```

    ```CommandPrompt
    # For Windows
        ./a.exe
    ```
3. For custom filename
    ```BASH
    # For Linux
        ./output_filename.out
    ```

    ```CommandPrompt
    # For Windows
        ./output_filename.exe
    ```
